Mobile Burger Menu

Slide-in overlay menu with hamburger-to-X animation, backdrop blur, and body scroll lock. Open on mobile (<768px) or resize browser to see.

Hamburger → X
Smooth transition from hamburger to close icon using CSS transforms
.burger.active span:nth-child(1) { transform: rotate(45deg) translate(5px, 5px); }
Slide-in Overlay
Menu slides from right: -100% to 0 with backdrop blur
.mobile-menu { right: -100%; transition: right 0.3s ease; } .mobile-menu.open { right: 0; }
Body Scroll Lock
Prevents background scrolling when menu is open
if (menuOpen) { document.body.style.overflow = 'hidden'; } else { document.body.style.overflow = ''; }
Backdrop Click-to-Close
Overlay backdrop closes menu on click
.menu-overlay { opacity: 0; pointer-events: none; } .menu-overlay.active { opacity: 1; pointer-events: auto; }
📱 How to Test
On desktop: resize browser to <768px width

On mobile: tap the hamburger icon (☰)

The menu will slide in from the right with a blurred backdrop.
